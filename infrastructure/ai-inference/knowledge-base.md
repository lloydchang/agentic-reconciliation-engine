# Cloud AI Knowledge Base

## Federated Learning
Federated Learning enables training AI models across distributed edge devices without exchanging raw data. Key concepts:
- Privacy preservation through local training
- Model aggregation techniques (FedAvg, FedProx)
- Applications: Mobile devices, IoT networks, healthcare data
- Challenges: Communication efficiency, heterogeneity, security

## Homomorphic Encryption
Homomorphic encryption allows computation on encrypted data without decryption:
- Fully Homomorphic Encryption (FHE) enables arbitrary operations
- Partially Homomorphic Encryption (PHE) for specific operations
- Applications: Privacy-preserving AI inference, secure cloud computing
- Performance trade-offs: Computation overhead, key management
